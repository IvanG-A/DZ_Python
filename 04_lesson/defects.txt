Дефект 1: Метод contains возвращает True при поиске пустой строки
Приоритет: Средний
Описание: Согласно документации, метод должен проверять наличие символа. Пустая строка символом не является, но метод возвращает True.
Шаги воспроизведения:
  utils = StringUtils()
  result = utils.contains("abc", "")
Ожидаемый результат: False
Фактический результат: True
